Student Grade Manager

A Java console application for managing student records and computing grades.
Built as part of my Java learning journey, applying core Object-Oriented
Programming principles to solve a real academic use case.

Core Features;
- Add and store student records (name, ID, scores)
- Automatically compute total and average grades
- Display a full list of students and their results
- Simple, clean console-based interface

Tech Stack;
- Language: Java
- Paradigm: Object-Oriented Programming (OOP)
- IDE: IntelliJ IDEA

 OOP Concepts Applied;
- Classes & Objects — Student class modelling real-world entities
- Encapsulation — Private fields with getters and setters
- Arrays & ArrayLists — Storing and iterating student records
- Methods — Modular logic for computing and displaying grades
